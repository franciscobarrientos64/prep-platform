PREP Core Platform
Infraestructura compartida. Sin esto, nada funciona. Se construye una sola vez y todos los módulos la consumen.
🔐 Autenticación y roles (Supabase Auth)
🗄️ Base de datos PostgreSQL (Supabase)
⚡ Tiempo real WebSockets (Supabase Realtime)
🌐 Frontend React 19 + Tailwind
🚀 Deploy Vercel (edge network)
🧾 Facturación: Nubefact API (SUNAT)
💳 Pagos: Culqi + Mercado Pago
📱 WhatsApp: Meta Business API
📧 Email: Resend
🤖 IA: Claude API (Anthropic)
🏷️ Multi-tenant por restaurante
🔒 RLS (Row Level Security) por tenant
📄 PDF: jsPDF client-side
🛒 Precios: VTEX API (Wong/Metro/Makro)
